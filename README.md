# B3 Brand Assets

- `wordmark-blue.png` — blue wordmark, transparent (headers on white)
- `wordmark-blue-hires.png` — hi-res blue wordmark
- `wordmark-intelligence-white.png` — white lockup + "An Intelligence Company" tagline (navy backdrops)
- `wordmark-clean-white.png` — white wordmark, no tagline
